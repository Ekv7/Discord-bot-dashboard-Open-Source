# Security Policy

## Supported Versions

Aktiv gepflegt wird die jeweils aktuelle Version auf dem `main`-Branch.

## Meldung von Sicherheitslücken

Bitte keine Security-Probleme öffentlich als Issue posten.

Melde Sicherheitslücken privat an:
- Discord: `discord.gg/nVVRwfPCrw`

Bitte wenn möglich mitschicken:
- kurze Beschreibung
- Schritte zur Reproduktion
- betroffene Datei/Funktion
- mögliche Auswirkung
